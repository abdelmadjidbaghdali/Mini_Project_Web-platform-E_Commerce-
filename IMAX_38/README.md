# IMAXTECH – PC Solutions

**IMAXTECH** is a modern **e-commerce web application** dedicated to selling PC components and gaming hardware.  
This project is developed for **educational purposes** and demonstrates the implementation of a complete online store with both user and administrative functionalities.
This project was developed as part of an academic mini-project, following modern web development principles and best practices.

The application follows an **MVC-inspired architecture** and adopts a **Neon / Gamer UI theme**, providing a clean, consistent, and immersive user experience.

---

## 🖥️ Project Overview

IMAXTECH allows users to browse PC products, filter items by categories, manage a shopping cart, and place orders online.  
Administrators can manage products, users, and orders through a dedicated admin dashboard.

The project focuses on:
- Clear and organized code structure
- Unified UI/UX across all pages
- Practical use of web technologies
- Pricing adapted to the **Algerian market (DZD)**

## 📦 Installation & Running the Project

Follow the steps below to run the project locally.

### 1️⃣ Install XAMPP
Download and install **XAMPP** from the official website:  
https://www.apachefriends.org/

---

### 2️⃣ Project Setup
1. Open the XAMPP installation directory
2. Navigate to the `htdocs` folder
3. Place the project folder (e.g. `IMAXTECH`) inside `htdocs`

---

### 3️⃣ Start Server Services
- Open **XAMPP Control Panel**
- Start:
  - **Apache**
  - **MySQL**

Ensure both services are running correctly.

---

### 4️⃣ Database Setup
1. Open your browser and go to:
http://localhost/phpmyadmin

2. Create a new database named:


IMAXTECH

3. Import the SQL file:


IMAXTECH.sql


---

### 5️⃣ Application Configuration
- Ensure the database connection file uses:
  - Database name: `IMAXTECH`
  - Correct MySQL port (e.g. `3306` or `3307`)
- Verify file paths according to your local setup

---

### 6️⃣ Run the Application
Open your browser and navigate to:


http://localhost/IMAXTECH/index.php


(Replace `IMAXTECH` with the actual project folder name if different.)
