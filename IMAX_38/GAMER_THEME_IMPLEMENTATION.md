## 🎨 Theme Customization

To modify neon colors, edit in `base.css`:

```css
:root {
  --color-bg: #0a0520;              /* Dark base */
  --color-primary: #00ffff;         /* Cyan neon */
  --color-accent: #ff00ff;          /* Magenta neon */
  --color-success: #00ff00;         /* Green neon */
}
```
